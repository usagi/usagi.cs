OptionalTReset メソッド (T, Boolean)

[<<Back to Home](https://github.com/usagi/usagi.cs/blob/master/Help/Home.md)
ボクシングオブジェクトにオブジェクトを新たに内包させます。

既に内包しているオブジェクトがあった場合その運命はガベージコレクターに委ねられます。たぶん。
