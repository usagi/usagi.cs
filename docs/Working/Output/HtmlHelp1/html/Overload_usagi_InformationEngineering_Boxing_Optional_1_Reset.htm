Reset(T, Boolean)
ボクシングオブジェクトにオブジェクトを新たに内包させます。

既に内包しているオブジェクトがあった場合その運命はガベージコレクターに委ねられます。たぶん。

Reset(T, T)
新たなオブジェクトを内包させると同時に既に内包しているオブジェクトがあった場合には取り出せる版の Reset　です。
