Enumerable
Enumerable をより便利に した列挙機能を提供する。

大別して2系統の Range が使用可能になる。

1. Enumerable 互換の

コピー
Range( Begin, Count )
系 2. 数学や工学で一般的な区間表現に互換の
コピー
Range( Begin, End, Termination )
系

後者は必ず引数を3つ以上とり、区間の始端、終端に続けて 区間端の開閉を EnumerableRangeTermination で要求する。
