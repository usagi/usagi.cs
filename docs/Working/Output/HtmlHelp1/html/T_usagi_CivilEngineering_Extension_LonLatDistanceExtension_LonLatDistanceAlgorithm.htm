public enum LonLatDistanceAlgorithm
メンバー
  メンバー値説明
Haversine0 Haversine 法（半正矢関数の公式）
  • 惑星の赤道方向への半径から概算する。
  • 惑星の楕円体としての扁平率は無視する。
  • 負荷は軽いが高精度な計算結果を求めたい場合には向かない。
Vincenty1 Vincenty 法
  • 惑星を楕円体近似した赤道方向への半径、回転軸方向への半径、扁平率を考慮し、十分な計算精度を反復計算により確保する。
  • 負荷は高いが高精度な計算結果を求めたい場合に向く。
