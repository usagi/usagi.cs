内包するオブジェクトがあれば value へ取り出し true を返します。

内包するオブジェクトが無ければ value には何もせず false を返します。

名前空間:  usagi.Boxing
アセンブリ:  usagi (in usagi.dll) バージョン: 1.1.0.0 (1.1.0.0)

	ref T value
)

パラメーター

value
型: T
内包するオブジェクトがあれば代入されます。無ければ何もされません。

戻り値

型: Boolean
内包するオブジェクトを value へ取り出せた場合は true 、そうでない場合には false が帰ります。
解説
Get や暗黙の型変換で例外が飛ぶ可能性を嫌う場合にどうぞ。
